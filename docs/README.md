AGENDA CULTURAL | Santiago Miguel Bentati | 2025
DESCRIPCIÓN DEL PROYECTO
Agenda Cultural es una plataforma web full-stack para la gestión y promoción de eventos culturales.

STACK TECNOLÓGICO
Backend
Node.js 20.x          Runtime JavaScript
Express 4.x           Framework web
TypeScript 5.x        Lenguaje tipado
JWT + bcrypt          Autenticación y seguridad
Nodemailer            Envío de emails
Express-validator     Validación de datos
Frontend
Angular 20.3          Framework SPA
TypeScript 5.x        Lenguaje tipado
CSS                   Estilos y diseño
RxJS                  Programación reactiva

Base de Datos
MySQL 8.x             Base de datos relacional
6 Tablas              Modelo normalizado 3FN
DevOps
Docker                Containerización
Docker Compose        Orquestación
Git + GitHub          Control de versiones
 
MÓDULOS DEL SISTEMA
Módulo de Autenticación
•	Registro de usuarios con verificación por email
•	Login seguro con JWT
•	Recuperación de contraseña
•	Gestión de sesiones
Módulo de Eventos
•	CRUD completo de eventos
•	Categorización múltiple
•	Búsqueda y filtros avanzados
•	Geolocalización con mapas
•	Sistema de destacados
Módulo de Flyers
•	Galería pública de flyers
•	Relación con eventos
•	Sistema de etiquetas
•	Contador de vistas
Módulo de Solicitudes
•	Creación de solicitudes de diseño
•	Workflow con estados
•	Sistema de prioridades
•	Calificación de trabajos
Panel de Administración
•	Gestión de usuarios y roles
•	Gestión de categorías
•	Aprobación de eventos
•	Gestión de solicitudes
•	Estadísticas del sistema
